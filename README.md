# AI Food Waste Reduction Assistant

## Overview
AI-powered Streamlit application that helps users reduce food waste through recipe recommendations, storage suggestions, waste reduction tips, and sustainability tracking.

## Features
- Recipe Assistant
- Storage Assistant
- Waste Reduction Tips
- Sustainability Dashboard
- Sustainability Score

## Technologies Used
- Python
- Streamlit
- Pandas
- GitHub

## SDG Goal
SDG 12 - Responsible Consumption and Production

## Benefits
- Reduces food wastage
- Saves money
- Promotes sustainable living
- Reduces carbon footprint
